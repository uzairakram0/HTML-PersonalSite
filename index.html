My journey in computer science began partially in seeking to understand and demystifying how computers work. My understanding of computers thus far is that a computer is a system of integrated circuits that can represent algorithmic logic; execute commands or instructions from memory in a specific order. It can represent information and boolean logic using binary states (either electrical current is passing through or not). It can also simulate mathematical logic and perform mathematical computations using adders, comparators, and other circuits. It also uses various digital logic circuits to implement digital functions: decoders, encoders, multiplexers and demultiplexers.
As i advance through my degree plan, I plan to focus on machine learning and artificial intelligence. I believe this focus will present a good challenge and will help me build an understanding of computation as it relates to cognition.
Computer Science however is not a study of computers but a study of computation. Computers are a tool for performing computations. Computation is methodology for producing imperative knowledge. Computation can be analogous to a mathematical function for a given set of inputs there is a defined output. Many professors say "Computation is both an art and a science" the creation of a computational procedure or algortihm is both a creative and a logical process. I think that computer science is a great field for me as it allows me to engages my mind in problem solving and critical thinking, which I enjoy.
